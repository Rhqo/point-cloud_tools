# Point Cloud Tools

## 1. FPS (Farthest Point Sampling)
```
  uv run run_fps.py <input_ply.ply> <target point number>
```

## 2. HDBSCAN Clustering
```
  uv run hdbscan_clustering.py <input_ply.ply>
```

## 3. Filtering
```
  uv run hdbscan_filtering.py <clustered_input_ply.ply>
```
